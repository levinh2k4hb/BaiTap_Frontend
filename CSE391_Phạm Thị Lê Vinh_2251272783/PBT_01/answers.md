# CÂU A1
**1.5 bước chính:**
- DNS Lookup (Phân giải tên miền)
- TCP Handshake & TLS Negotiation (Thiết lập kết nối & Bảo mật)
- HTTP Request (Gửi yêu cầu)
- Server Response (Máy chủ phản hồi)
- Browser Parsing & Rendering (Phân tích và vẽ giao diện)


**2. DevTool của Chrome**
Tab Network là trung tâm phân tích mạng. Nó ghi lại tất cả các request (yêu cầu) mà trình duyệt đã gửi đi để tải trang web, cũng như dữ liệu nhận về.
